# CodeGeeX

#   欢迎使用我的chat dome应用

这是一个简单的演示应用程序。

## 使用说明

1. 输入您的问题
2. 等待回复
3. 享受对话！

## 功能

-  支持多轮对话
-  支持上传本地zip压缩包项目，可以进行项目问答和对项目进行修改

## 安装

1. 克隆仓库到本地
2. 安装依赖：`pip install -r requirements.txt`
3.  运行应用：`chain run run.py --port 8888`

## 注意

请确保您的网络环境可以访问CodeGeeX的API。
